GreenEd Login Page

GreenEd is an amazingly cool and innovative mobile app with whole lot of advanced features for schools to share every details of students with their parents. GreenEd helps to establish a great connection between schools and parents to develop brighter kids for future. Parents can see every detail of their child on there fingertips anytime anywhere.

HeadMaster Wall Page

A common wall page for all users including hm, teachers and parents with posting access for hm and teachers, they can post image as well as text in wall posts. Teachers and hm can share tips and thoughts for parents to upbring their kids to think out of box and achieve great skill set for the modern world. Parents can view the posts submitted by hm and teachers and they can like or dislike the posts based on the content.

HeadMaster Profile Page

Different logins for headmaster, all teachers and all parents.... A main menu on the left hand side of the app with user image and welcome username and below that contains the menu item which provide access to pages with diiferent features in the app.

Dashboard Page

Dashboard displays the performance of the school to hm, performance of the class to class teacher, performance of subjects to subject teacher and performance of his/her kids to parents. By default it displays the result of the latest examination. There is a filter on the top right corner, using that filter a user can choose previous examinations of current year as well as results of previous years.

Message Page with Chat

Message page contains two tabs namely chats and contacts. A user can initaite a chat from selecting an other user from the contacts tab. Already initiated chats can be viewed on the chats tab. All chats initiated will be listed on the chats tab and the user can engage in those chats again by simply clicking it.

HM Dashboard – Subjects Pass/Fail Tab – Grades and Class Toppers

Pass - Fail : Hm Dashboard contains details of total number of students pass/fail from the entire school. He/She can click on the chart to find the list of students passed as well as the list of students failed. From that list he/she can click on any particular student to get all his academic details. He/She can also use the search bar to find a particular student from the long list of students and this helps not to scroll down the entire list to find a student. Classnames are displayed along with student name so that it can help to find students from a particular class with out any confusion even if there are few other students with same name in the school. From these list he/she can use the back button on the top left corner to go to the previous page.

Teacher Dashboard

Pass - Fail : Teacher Dashboard contains details of total number of students pass/fail from the subjects taken by the particular teacher. He/She can click on the chart to find the list of students passed as well as the list of students failed. From that list he/she can click on any particular student to get all his academic details. He/She can also use the search bar to find a particular student from the long list of students and this helps not to scroll down the entire list to find a student. Classnames are displayed along with student name so that it can help to find students from a particular class with out any confusion even if there are few other students with same name in the school. Parent Dashboard

By Default it contains the marks and details of the latest exam. You can see all the marks and details of previous exams by simply clicking and selecting the appropraiate exams from the filter on the top right corner of the app.
Subject Marks: A bar chart displaying marks taken by student on each subject with the pass marks displayed on blue bar and fail marks displayed on red bars.
Attendance : A pie chart displaying the number of days present in blue color and number of days absent in red color.

Parent Dashboard Details

Total number of marks taken by student in all subjects is shown here.Grade taken by student is shown here.Status of student whether he/she is pass or fail is shown here.Class in which student is studying is shown here.Below that you can find a link to see the overall dashboard of the student

Overall Dashboard – Subject Marks Line Graph

Subjects : A line graph is shown to display all individual subjects marks taken by student on each exam conducted on the current academic year. Each subject is shown with an unique color and by clicking on the subject name given below you can disable and enable the subject line graph for every subject.

Bus Tracking :-

This is an exclusive feature available only for parents to view the exact location of the bus and can also see the approximate time by which the bus will reach on their kid's pickup point. They can also check whether the bus reached school by time and also can know at any time where the bus exactly is. Parents can track the location and movement along with approximate of time for schoolbus arrival effortlessly using GreenEd Mobile App.

FavoriteTeacher Page

Favourite teacher is a page available only for parents to select a favourite teacher from the list of teachers who were teaching there kids. They can simply swipe or use like or dislike button to select a favourite teacher. They can swipe towards left to disliking and swipe towards right to select the favourite teacher from the list of all teachers provided. This is an exclusive feature to select the favourite teacher. They can change a previous selection by selecting some other teacher as favourite teacher any time they wish.
