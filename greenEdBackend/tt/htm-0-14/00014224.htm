The Book Of Wisdom template is the perfect template for your next education / school website.

The idea behind this template was to create something that could not only be used for an education website but could easily be modified for any other kind of website.

The downloadable archive contains two "template" folders: "Full Template" & "Basic Template".

The full template has everything incorporated into it, whereas the basic template simply has the very basics to use the template as a base template, without having to remove all of predefined HTML code that demonstrates what can be accomplished with the template.

The template comes with a choice of:

  • Responsive - Full Width
  • Responsive - Boxed
  • Default light skin
  • 5 Predefined colour schemes

Page layout types include:

  • Standard - 1, 2, or 3 columns
  • Portfolio
  • Gallery
  • Blog Overview
  • Blog Single

In use with the framework all of the above layouts can be styled using: Full Width, Sidebar Left, Sidebar Right, Sidebar Left & Right

Extras Included in the downloadable zip:

  • All of the Layered PSD and PNG files
  • PHP + jQuery Contact Form

Predefined HTML Pages included:

  • Homepage
  • About Us
  • Team or Staff Overview + Single Team or Staff Member
  • Course Overview + Single Course + Weekly Timetable
  • FAQ
  • Archives
  • 404 Error Page
  • Basic HTML Sitemap
  • Contact

The template also contains our standard elements: buttons, accordions etc. which have been modified specifically for this layout.
